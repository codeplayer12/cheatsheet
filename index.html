HTML Summary Cheatsheet

Tag Name Description
<head> head contains machine-readable information
<title> title defines the document title
<link> link specifies relationships between the current document and an external source
<body> body represents the content of the html document
<article> article represents a self contained composition in a document,page,application or site
<aside> aside represents a portion of the document whose content is only indirectly related to the document's main content
